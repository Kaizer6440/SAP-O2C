# SAP O2C Simulator (Order-to-Cash)

## Overview
This project simulates the SAP SD Order-to-Cash (O2C) process using a web-based SAP GUI-like interface.

## Process Flow
- VA01 – Sales Order Creation  
- VL01N – Outbound Delivery  
- VL02N – Post Goods Issue  
- VF01 – Billing  
- F-28 – Incoming Payment  

## Features
- SAP GUI-like interface  
- Step-by-step transaction flow  
- Dynamic status updates  
- No external dependencies  

## How to Run
1. Download the project  
2. Open `index.html` in any browser  

## Tech Stack
- HTML  
- CSS  
- JavaScript
